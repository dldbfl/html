table: 표형식의 데이터
	tr		: table row 행
	td		: table data, 열
	th		: table header, 열의 제목
	
	< table border="1">
	테이블의 테두리를 생성한다. border 속성의 기본 값은 1이고
	그 이외의 값은 실행은 되나 html5에서의 표준은 아니다.
	
	table의 테두리 속성은 style에서 설정한다.
	 table{
	 border	: 10px solid blue;
	 border-collapse : collapse;
	 }
 	table의 td 테두리 합치기
 		border-collapse: collapse;
