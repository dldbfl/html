fieldest : 요소들의 그룹핑에 사용
        lavel    : 화면에 표시되는 일반문자
        display  : 화면에 표시되는 속성을 설정
                    inline      : 줄바꿈 안하고 가로세로 길이조절 할 수 없음
                    block       : 줄이 바뀌고 길이조절 가능
                    inline-block: inline 속성으로 줄은 안바뀌지만 block속성으로 가로길이 설정 가능
                    none        : 화면상에 보이지 않도록 한다.
        <input type = range onmousemove="getvalue()" onchange="getvalue()">
        onchange는 크롬에서 onmousemove는 익스플로러에서 실행가능
        //id가 inrang인 요소에 접근, 값을 가져온다.getvalue
